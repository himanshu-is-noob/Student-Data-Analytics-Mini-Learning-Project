# Student-Data-Analytics-Mini-Learning-Project
Student Data Analytics Learning Project

This is my one of the learning Project of Data Analytics . In this I tried to analyze student marks dataset and the various factors affecting the student marks .
In this Dataset I have three Subjects i.e Maths , Learning and Writing . I have various student details such as parental education , Study hours , parental marital status etc which I tried to analyze using Python , Numpy , Pandas , MatplotLib , Seaborn .
